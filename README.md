# Message Throtlling
Este repositorio contiene la lógica para el sistema de gestión de SMS, integrando Node.js, RabbitMQ y Redis.

## Requisitos
- Docker y Docker Compose
- Node.js instalado

## Guía de Inicio

Para poner en marcha el sistema, sigue estos pasos en orden:

1. Levantar la infraestructura (Docker):
   Comando: docker-compose up -d
